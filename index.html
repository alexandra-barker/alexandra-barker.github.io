Particle Wafer Defect Classifier

Transit in Beijing

Geospatial analysis of transit methods and commuting activity for GPS users in Beijing between 2007 and 2012. Focus on geospatial data visualization and web design.

  • More

Spotify Popularity Predictor

Machine learning model development of Spotify music library, creating a classification of "popular"/"unpopular" songs based on audible features. **Please note this project was completed prior to starting my Master's degree and as such contains many rookie mistakes.
